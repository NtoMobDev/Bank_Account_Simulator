# Bank_Account_Simulator
A simple Kotlin console application that simulates basic banking operationsoperations such as creating accounts, depositing, withdrawing, transferring funds, and viewing transaction history.Built using object-oriented programming and Kotlin best practices.

#Features
Create new bank accounts
Deposit and withdraw money
Transfer funds between accounts
View account details and transaction history
List all existing accounts

Input handled through console (readln())

#Concepts Used
Kotlin OOP (classes, objects, data classes)
Collections (List, MutableList)
Null safety & error handling
Functions and scope
Control flow (when, loops)
